Blockgame

This is a simple Minecraft clone written in C++. I wrote this as a programming challenge and I don't really intend on expanding or continuing it for now.
I might rewrite it in a different programming language (Rust???) and then add some extra features to it in the future though.

Controls:
- WASD to move around
- Space to jump
- Right click to place blocks
- Left click to break blocks
- 1,2,3,4,5,6,7,8 to select blocks
