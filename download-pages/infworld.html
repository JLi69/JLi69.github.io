Infworld

This was an OpenGL demo where I attempted to make a procedurally generated infinte world. The terrain is generated using Perlin Noise. Most of the time for this project was spent attempting to optimize the terrain rendering. Overall, I learned quite a few things from this project and I'm quite happy with how it looks.
