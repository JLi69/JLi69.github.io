The Server Must Scale

Made with gldeA and cfinn

This is the game that gldeA, cfinn, and I worked on for the 2024 GMTK Game Jam and we made it over the course of 96 hours. The theme of the jam was "built to scale". We decided to interpret it as "scaling up" in software development where you have to build a program to be able to handle more and more data and we figured a good representation of this would be a factory style game where you had to manage a server farm and build it to be as efficient as possible in order to keep up with the ever increasing amount of input data. Overall, it was very fun to work on this and I liked how the end result looks.

Controls:
- W, A, S, D = pan camera
- Scroll = zoom in/out
- Left click = place tile
- Right click = rotate tile
- Shift + Right click = reverse rotation
- Space = Pause the game (You can still build in pause mode)
- Escape = Quit session
- Q = deselect whatever tile you currently have selected
