Voxelworld

This is a voxel engine that I programming using OpenGL and the Rust Programming language. It's basically a Minecraft clone at the moment and it's still a work in progress at the moment. Think of it as a continuation of a previous project of mine, blockgame, which was a minecraft clone written with C++. You can browse the source code here.

Controls:
- W, A, S, D to move
- Space to jump
- Right click to place blocks
- Left click to destroy blocks
- Shift to sprint
- Tab to open block menu
- E to open inventory
